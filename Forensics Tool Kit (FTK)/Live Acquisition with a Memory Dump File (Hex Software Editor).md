

**Live Acquisition with a Memory Dump File**

 **Open the Memory Dump:** 
   * Go to **File** > **Open**
   * Select **mamdump.mam** 
   * Click **Open**

 **Search for "password":** 
   * Click the **Find** icon
   * Ensure **String** is selected
   * Type **password**
   * Click **Find**
